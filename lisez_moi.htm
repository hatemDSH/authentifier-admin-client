- : : : LES INSTRUCTIONS : : : -

 

PREREQUIS :

- APACHE et PHP
- MySQL

EasyPHP fait largement l'affaire (téléchargez-le ici).

NB: Les sessions doivent être activées dans le php.ini et un répertoire permettant leur stockage doit être spécifié.

 

INSTALLATION :

- Placez la totalité des fichiers du script sur un répertoire de votre serveur web

- Vous allez maintenant créer une BDD nommée 'DBprotect' (ou autre si vous voulez) à l'aide de PhpMyadmin (par exemple)

- Éditez au besoin les paramètres de connexion du fichier connexion.php (pensez y si vous utilisez un autre nom pour votre base)

- Exécutez La commande SQL contenu dans le fichier dbprotect.sql

- Voilà ! Ouvrez maintenant le fichier index.php via votre navigateur web avec le chemin d'accès complet depuis votre serveur Apache.

- Il ne reste plus qu'à vous connecter avec les identifiants de l'utilisateur par défaut :

login : "toto"
mot de passe : "mdp"

La suite vous sera expliquée une fois connecté. Vous allez pouvoir ajouter des utilisateurs et éditer des actions en fonction des niveaux d'accès. N'oubliez pas de regarder les sources pour comprendre le fonctionnement.

 

POUR ALLER PLUS LOIN :

N'hésitez pas à éditer ces quelques pages sous Dreamweaver ou tout autre WYSIWYG pour en modifer l'apparence et afin de le personaliser à votre goût ou afin de respecter la charte graphique du site sur lequel vous l'implanterez.

Pour une meilleure sécurité, n'oubliez pas que seule une connection sécurisée SSL offre un cryptage des données qui circulent sur le réseau. Je ne saurai donc trop vous conseiller d'utiliser ce script de pair avec une connexion sécurisée.

N'oubliez pas non plus de supprimer l'utilisateur par défaut (toto) de votre base de données, sinon, à quoi bon avoir un coffre fort si on en laisse la clé à la porté de tous...
